# hvtiRtables 0.1.0

## New features

- Initial release. `manuscript_flextable()` converts a `gtsummary` table
  object into a Blackstone-compliant `flextable` (flat header, no merged
  cells, house font/rounding rules). `save_manuscript_table()` writes it to
  a `.docx` with footnotes and an abbreviation key as text below the table.
